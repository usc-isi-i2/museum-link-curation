About

This web app was developed to curate linked data of "American Arts Collaborative".

info_outline

Curate

Curate pairs of data from two different sources by selecting "YES", "NO" or "NOT SURE" button along with writing optional comment about your response. Pair will have matched and unmatched attributes put side by side for easier comparison.

info_outline

Profile

Update your preferred museums, track your progress and configure other profile information. You can also see your curation statistics with respect to different museums and also compare it with total curation statistics.

info_outline

Results

Browse, search and filter already curated data. You can also export already curated pairs of data in json format. You can select one or more museums and flags (Matched, Unmatched, Not Sure) and click export to download the json file. This is helpful to do further analysis on top of curated data.



All of museum data has been sourced from American Art Github, cleaned and converted to RDF with the help of Karma using cidoc-crm based reference data models at AAC Review. RDF data is also available to be queried at Sparql endpoint here. Curation interface is populated with pairs of data which is generated using suitable record blocking and linking techniques. Code for this web app is available at Github. You can contact us at nilayvac@usc.edu for any issues, bugs, feedback or feature requests.
